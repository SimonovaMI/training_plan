Инструкция для администратора
1.	Установить MySQL
2.	Установить MySQLWorkbench
3.	Создать подключение
4.      В web_application.web_application.__init__.py прописать логин и пароль
5.	Создать схему training_plan
6.	Запустить manage.py 
7.	manage.py makemigrations
8.	manage.py migrate
9.	manage.py createsuperuser
10.	Создать суперадмина
11.	Запустить приложение 
12.	Перейти по ссылке http://localhost:8000/admin
13.	Создать фитнес-клуб
14.	Создать зоны
15.	Создать группы
16.	Создать роли
17.	Создать пользователя(из него создать клубную карту)
18.	Создать расписание
19.	Создать планируемое посещение зоны клуба
20.	Создать планируемое посещение группового занятия
21.     Зайти по ссылке http://localhost:8000/accounts/login
